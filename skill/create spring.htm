 create spring(造泉術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
造泉術 create spring 80 5 0 0 22 0 0 20 8
　
簡介

　

造泉術，這個法術會製造出水源。

　
　
說明

　

冒險者系法術，為新手練等必備的技能，作用為製造一個飲水機出來，此物品不能被移動，也不能被拾取，所以某些地方有怪物會亂撿東西的話，可以用此技能當做路標。重點是，他還可以解渴！其功用與製造食物術( create food )一樣，但不可以自由命名！

　
　
前置技能 造水術(create water)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座冒險者巨像聳立在這兒./player statue
