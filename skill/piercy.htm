 piercy(刺瞎)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
刺瞎 piercy 0 100 35 0 0 45 0 0 5
　
簡介

　

刺瞎，此技巧能讓你在戰鬥中以手中可行刺之武器刺向敵人的雙眼而造成敵人的失明。

　
　
說明

　

盜賊系技能， 可以使目標看不見，且此技能的作用時間可以重新計算。近期幾年來，許多玩家常利用此技能來打一些大型的怪物，或是作為量產子彈的方式。這技能只能靠去除魔法術( dispel magic )來解除其技能效用，因此算是一個非常好用的輔助性技能。

【註】作用時間重新計算是指，當被刺瞎消效果快消失時，再補刺一下又會回到原有的時效。

　
　
前置技能 背刺(backstab)
相剋技能 無
