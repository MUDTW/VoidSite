 confuse(困惑術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
困惑術 confuse 241 30 0 0 45 0 0 50 10
　
簡介

　

困惑術，此一法術將可使敵人的思路受阻，因而智力減弱。

　
　
說明

　

巫師系法術，效果為降低目標智力值( int )，但只能夠上一次，固定降低四點，其成功率較低，但若搭配魔法沙漏( magic clock )使用，可以將目標的智力值降到最低。可以用來輔助一些需要降低對方智力後才能使用的技能。

　
　
前置技能 博識術(lore)
相剋技能 無
