 pass door(遊離化)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
游離化 pass door 74 130 0 0 45 0 0 45 7
　
簡介

　

游離化，這個法術可以使得施法者可以穿過一些關著的門。

　
　
說明

　

巫師系技能， 很實用的法術，可以讓玩家直接穿越一些已經被關上、鎖住，或是根本可不了的門。一般都是利用學有煉符咒( makescroll )的道士幫忙生產符咒後使用，或是使用一些擁有此功能的藥水或道具。而有些道具及怪物，更是需要運用此法術才能過去看到他們。

※特性

  • 穿越與否由該區域作者設定，請玩家自行試驗
  • 若是一扇門你穿越過去，但是過去後馬上進入戰鬥且沒有任何出口，又因為無法開啟四週的門(沒有鑰匙)，此時玩家將無法逃跑，但是若玩家習有遁逃術( abdcond )，則可以指定從剛剛通過的門，或是其他方向的門逃跑，這點請特別注意。

　
　
前置技能 全隱術(pass door)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座力場系巫師巨像聳立在這兒./wizard statue
　 Ulitima - (白色聖光) 魔法師瑪莉安(mariah mage)正在找尋著不知名的典藉.
