 outdo waver(凌波微步)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
凌波微步 outdo waver 0 300 25 0 0 63 27 0 20
　
簡介

　

凌波微步，為逍遙派之獨門輕功，練此功者不但身輕如燕而且將可在戰鬥之中快速的改變身法，令敵人捉摸不定。可以閃躲過怪物的特殊攻擊。

　
　
說明

　

防禦系最終技能，學習這招有一定的難度，但是非常的值得，不論是法術、特殊攻擊、一般攻擊、技能攻擊、甚至是大神( Imm )的攻擊，都能夠閃的一乾二淨，而且閃掉的同時，只會消耗掉玩家的移動力，是一個非常實用、變態，非常值得去練一次的技能，遊戲內更有一些是為了此技能而打造的職業。此技能閃躲機率由躲避能力( dc )值決定，躲避能力越高則閃掉的機率越高。而發動機率較高數值是在躲避能力值達1800 點以後。另外敏捷度( dex )也會決定閃躲的機率(可以影響躲避能力高低)。

※特性：

  • 施展此技能時，固定消耗移動力 300 點。
  • 凌波微步在閃躲攻擊有二種敘述，而這二種敘述決定了花費移動力與否
    一是閃掉「攻擊」：如技能、法術、一般攻擊，都屬於此類，閃躲此類攻擊消耗掉移動力 50 點。
  • 二是閃掉「特殊攻擊」：如怪物所使用的特攻就是屬於此類,此類型的攻擊，閃躲不會消耗移動力。
  • 若移動力過低，則無法閃躲攻擊。
  • 躲避能力的的高低請忽略那個「 - 」值，純粹以數字大小決定高低。

　
　
前置技能 輕功(agility)
相剋技能 無
