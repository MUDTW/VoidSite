 swordmaster(劍訣)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
劍訣 swordmaster 0 300 25 0 0 0 90 0 0
　
簡介

　

劍訣，為擅用劍者最高之心法，習得此法後，方可將自身所學之劍法發揮到爐火純青之境界，而且亦將劍人合一，不容易被人擊落手中之劍。

　
　
說明

　

劍士系基礎技能，可以使低階劍招威力增強數倍，並且可以使被擊落武器機率大幅度地降低，可以說是卸除武器( disarm )的剋星。

　
　
前置技能 無
相剋技能 雙手持劍法(two weapon)、閃電奇襲式(blitz)、龜派氣功(kamekameha)
　
習得處 雕像館 - (白色聖光) 一座劍士巨像聳立在這兒./swd statue
　 雕像館 - (白色聖光) 一座魔劍士巨像聳立在這兒./mag-swd statue
