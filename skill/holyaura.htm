 holyaura(大至聖光芒)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
大至聖光芒 holyaura 217 30 0 0 198 0 0 200 10
　
簡介

　

大至聖光芒，此一法術將會對施法者所在團體內之所有隊員披上聖光，為一整體性法術。

　
　
說明

　

牧師系技能，為聖光術( sanctuary )的範圍技，能替同一房間內的隊員施上，且於戰鬥中可使用。

　
　
前置技能 聖光(sanctuary)、法力加持(manamaster)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座教宗巨像聳立在這兒./cleric statue
　 Ulitima - (白色聖光) 大不列顛的統治者--英王(Lord British).
