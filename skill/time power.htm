 time power(子午大法)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
子午大法 time power 0 800 0 50 162 0 0 0 0
　
簡介 　

子午大法，為修道之人的上乘心法，習得此心法之後，將可在正午或子夜之時吸納天地日月之精華，並化為己身之精神力。

　
　
說明

　

道士系技能，有了他才可以學習其他高階道士系技能。另外，習有此招的玩家可於特定時間補滿其精神力。

※特性：

  • 若是心地陣營為善良的玩家，在室外且時間為「AM 12:00 」可以補滿精神力。
  • 若是心地陣營為邪惡的玩家，在室外且時間為「PM 12:00 」可以補滿精神力。
  • 若是心地陣營為中立的玩家，在室外且時間為「AM 或 PM 12:00」可以補滿精神力。
  • 要獲得此一效果的前提，是本身精神力不可以大於等於上限值。
  • 要練此技能，可以到地藏廟練。

　
　
前置技能 雲遊術(teleport)、加強武器術(enchant weapon)
相剋技能 魔力加持(spellmaster)、法力加持(manamaster)
　
習得處 雕像館 - (白色聖光) 一座道士巨像聳立在這兒./tat statue
