 windslice(風嘯之斬)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
風嘯之斬 windslice 0 300 75 315 0 270 0 0 50
　
簡介 　

風嘯之斬，一套堪足以媲美風之靈動的劍技，運用全身的力量，一次爆發出的八次攻擊。使用後將會耗盡體力，但也由於體力不支，必須要延遲一段時間方能再次出手。

　
　
說明

　

騎士系技能最強攻擊技，會在瞬間打出八段攻擊，但出招很慢且協調性很高，是一種需要知道使用時機的技能。如果左手未持武器也可以打出一樣多段數的攻擊，但左手的攻擊力會減少，出招時右手一定要持有武器(鎗炮、弓箭類不行)。

※特性和使用時機：

  • 會受到攻擊命中率( hr )的影響，而且影響很大，越高就越容易打出多段數攻擊，但這也是屬於機率性的。
  • 攻擊力依攻擊傷害率( dr )及武器攻傷值而決定。
  • 攻擊時，如果武器有附加的隱藏法術，也會出現。
  • 遇到躲避能力較高的敵人，就很容易被閃躲過。
  • 乾坤大挪移( refl )是這技能的剋星，會被反彈回來。
  • 遇到很會格擋( parry )和左手格擋( block )的敵人，就請多多利用這招。
  • 魔法結界( barrier )及凌波微步( outdo waver )，也很容易檔掉這招。
  • at at wind，這個wind就是指風嘯之斬，遊戲中騎士最常用到的連續攻擊指令。

簡單來說就是屬於防禦技的技能內，除了格檔系的以外，這招幾乎都會被克制到，而且因為副作用很強，所以一般都是用來當做最後的一擊。此外遇到很會逃跑的敵人，因為是多段攻擊，所以追殺還滿累的。

　
　
前置技能 雙手持劍法(two weapon)、大地之斬(earthslice)
相剋技能 無
　
習得處

　

 騎士堡 - (白色聖光) 一位傳說中偉大的騎士亞瑟王(Arthur)站在你的面前。
