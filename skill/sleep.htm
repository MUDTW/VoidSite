 sleep(催眠術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
催眠術 sleep 38 40 0 0 67 0 0 70 8
　
簡介

　

催眠術，傳說中黑魔法，利用自身強大的精神裡使目標強制進入睡眠狀態。

　
　
說明

　

巫師系法術，十分實用的技能，可以使目標睡著，如果玩家中招，被一些攻擊打到，威力會增加，其次當玩家處於睡眠狀態時，可以在他身上，上很多負面法術，而不會被回擊，是一個很偏向 pk 方面的技能。如果施法於怪物身上，可以使他一直處於睡眠狀態，也可以分開正在打架中的怪物。此法術施放機率與智力( int )無關，但只能施放在等級小於等於施術者的怪物或玩家身上。

　
　
前置技能 詛咒術(curse)
相剋技能 無
