 spellmaster(魔力加持)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
魔力加持 spellmaster 0 850 0 0 405 0 0 0 0
　
簡介

　

魔力加持，此技能乃魔法師在施用多次魔法之後所領悟到的，習得此技之後魔法的威力將更加強大，且花費較少之精神力。

　
　
說明

　

巫師系技能，是輔助法術闇黑結界( dark space )和防禦性法術魔法結界( barrier )的前置技能。另外習得此招之後，可以增加魔法強度，因此可以使一些攻擊性法術威力提高。還可以使施法時的消耗的精神力減少，對於巫師系職業來說，是一個需要思考到角色類型，再決定學習與否的技能。

　
　
前置技能 無
相剋技能 法力加持(manamaster)、子午大法(time power)
