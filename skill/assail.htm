 assail(奇襲術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
奇襲術 assail 0 140 35 72 0 45 0 0 12
　
簡介

　

奇襲術，給予敵人出奇不意的攻擊，雖然攻擊力不若背刺般嚴重，但是優點是戰鬥中仍可施展此技。

　
　
說明

　

盜賊系技能，為背刺( backstab )的替代技，取代背刺無法在戰鬥中的攻擊的缺點，可是威力低背刺一半以上，與一般技能使用方式相同。如果對背刺後想繼續接攻擊的玩家來說是必學技能。一般較少人會學，因為小偷打怪本來就不是主流，通常都是用於做 pk 角色，而 pk 時只需要背刺，就已經能決定生死。

　
　
前置技能 背刺(backstab)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座小偷巨像聳立在這兒./thief statue
　 吉雅尼斯王城 - (藏匿) 穿著一襲黑衣的小偷(thief).
