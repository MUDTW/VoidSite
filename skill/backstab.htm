 backstab(背刺)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
背刺 backstab 0 900 55 405 0 90 0 0 14
　
簡介

　

背刺，從敵人的背後使用致命的背刺，這項技能會使你的對手遭受到極為可怕的傷害，尤其學習過家強背刺( BS POWER )的人，更能發揮背刺的威力，在你的對手睡著或是看不見你的時候，背刺將具有更強的殺傷力。

　
　
說明

　

盜賊系攻擊技能，因為以下兩個原因，所以背刺用來打人很強，但打怪就極為不實用。
　1.背刺在戰鬥中無法使用，遇到會主動攻擊的怪物，即沒辦法以背刺攻擊。
　2.背刺的附加效果為延遲行動，但此效果對怪物沒有用
因此此技能只有在 pk 場跟玩家對打時比較實用，且是個超好用的 pk 技能。

※特性

  • 學習加強背刺( bs power )後，威力可增加約 60~70%。
  • 若是目標處於睡眠狀態下，背刺的威力將多增幅約 80~100%。
  • 若是目標敵人看不見你時，也有加乘效果，這裡可分為兩種情形
    A.目標處於攻擊目標若是屬於「瞎」的狀態下，則威力增加約 10~20%。
    B.如果自身有隱身術( invis )或藏匿( hide )，且攻擊目標看不到，則增加威力約 10~30%，且只會計算其中一個，沒辦法累加威力，即隱身術( invis)或藏匿( hide )皆有時，只計算其中一個。
