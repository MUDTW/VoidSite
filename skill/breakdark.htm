 breakdark(破闇黑結界)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
破闇黑結界 breakdark 0 0 0 0 0 0 0 500 0
　
簡介

　

破闇黑結界，透過一定的智力，了解了闇黑結界的紋理後可以將其吸收化為烏有。

　
　
說明

　

巫師系技能，為了純巫師而設計的專屬技能，可以在施展闇黑結界( dark space )失敗後，反彈自身時，使用此技能脫離困境，但使用時得花費大量的精神力，且失敗率極高，但因為不消耗記憶點術，所以為純巫師系巫師必學。

　
　
前置技能 闇黑結界(dark space)
相剋技能 加強破壞(enhanced damage)
　
習得處

　

 神秘高塔 - (白色聖光) 歐尼爾大陸的黑魔法女王夏雅(mistress Shaya)正站在這.

【註】需解完謎題 - 夏雅的煩惱。
