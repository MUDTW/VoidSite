 hunt(獵殺)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
獵殺 hunt 0 250 25 225 0 90 0 0 8
　
簡介

　

獵殺，使用此技可以在對方脫逃時，馬上追趕過去斬草除根。當被你獵殺的敵人走到你所在的房間之時，你也會主動的開啟戰鬥，並且先行出手。

　
　
說明

　

騎士系技能中的一個特殊技能，可以在目標脫逃時，馬上跟著他跑然後繼續攻擊。或是在目標跑掉後，被你遇上或遇上你時，可以強迫他進入戰鬥中，而且可以優先攻擊他。但如果獵殺目標看不見你，此技就會被破解，雖然效果還是在，但是會因看不見你而無法優先攻擊，可使用在一些非常會逃跑的敵人身上。但如果對方的攻擊力很強(反擊很痛)，或是會主動攻擊玩家mob較多的地方，使用這招往往都會變成自殺。另外追殺時，追隨目標到另一房間時，本身的協調性並不會重新計算，故使用前請三思.........

【註】如果直接輸入hunt而不加目標名稱的話，就是解除獵殺狀態。
　
前置技能 三段突刺(third attack)
相剋技能 閃電奇襲式(blitz)
