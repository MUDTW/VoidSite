 recharge item(補充術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
補充術 recharge item 213 30 0 0 54 0 0 50 10
　
簡介

　

補充術，使用這個法術，可以補充一些具有法力裝置物品的法力，不過稍一不慎則會將物品毀損。

　
　
說明

　

冒險者系法術，算是很實用的法術，遊戲中有一些道具，穿上後揮動會有法術效果，而這些法術如果快用完時，就是使用此技能去補充使用次數。尤其現在有非常多的法術都能夠使用道具附加，所以此法術的重要性也越來越高。此外，在施展時，有機率會使被補充的物品爆炸(物品會消失)。

　
　
前置技能 鑑定術(identify)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座冒險者巨像聳立在這兒./player statue
　 Ulitima - (白色聖光) 身上發出萬丈光芒的時空之王(timelord).
