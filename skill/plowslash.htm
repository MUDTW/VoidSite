 plowslash(天罡北斗陣)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
天罡北斗陣 plowslash 0 50 60 135 0 0 225 200 8
　
簡介

　

天罡北斗陣，為全真教陣教之寶，各由七名熟悉此劍陣的人組成，依照北斗七星之位排列，引用蒼穹星空無比的力量，重擊陣中的敵人，此陣一發動，則必須待陣形解散後方可停止。

　
　
說明

　

道士系攻擊技能，也是目前四度空間中威力最大最可怕的技能。此技威力由使用者的攻擊傷害率( dr )來決定。但一定得由「七名同一隊伍的玩家」於「同一房間」且「內力值足夠、協調性在一定值以下」方可出招，限制條件許多，通常沒有任何一個怪物可以捱過此技兩次攻擊。而且此技能施展後，必定要將七下劍技擊出才會結束，與天馬流星拳( cometfist )和風嘯之斬( windslice )不同，即不論怪物逃走與否，攻擊一樣會持續直至七下擊出。但因此技施展所需人數較多，因此很少人習有此招。有興趣的玩家可以約一約，一起練，不會讓你們後悔的！
