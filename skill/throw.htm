 throw(武器投射)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
武器投射 throw 0 50 25 27 0 0 0 0 6
　
簡介

　

武器投射，學得此技巧之後可將手中的「武器」向敵人丟射，所造成的傷害力，依各人的攻擊傷害率( dr )來計算，但投射出去的武器將可能會消失不見，切記 ！

　
　
說明

　

盜賊系技能，可以將右手的武器丟向敵人來造成攻擊，威力決定於施招者的攻擊傷害率( dr )，攻擊傷害率越高，則威力越大。只是丟出去的武器會隨著攻擊一起不見，是一個冷門非常久，也沒有什麼人想用的技能。

　
　
前置技能 無
相剋技能 無
