 locate object(定位術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
定位術 locate object 31 10 0 0 18 0 0 20 9
　
簡介

　

定位術，這個法術將會顯示出你所要找的物品的位置(在誰的身上)。

　
　
說明

　

冒險者系法術，算是很實用的法術，可以藉由此法術來尋找物品，或是利用尋找物品的功能，來判斷擁有此物品的怪物是否重生。對解謎還有找東西的幫助非常大。

　
　
前置技能 無
相剋技能 無
