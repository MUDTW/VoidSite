 makescroll(煉符咒)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
煉符咒 makescroll 0 30 0 0 0 135 0 0 12
　
簡介 　

煉符咒，這一個失傳已久的茅山術，源自於中國古代道士，可以將所學的法術藉由畫符術，記錄於符咒之上，你可以使用 Recite 這個指令來使用符咒。

　
　
說明

　

道士系特殊技能，非常實用的技能之一，可以將自身所學習過的法術，畫於一張卷軸上，然後交由他人朗誦，就可以施展出此法術。每一次煉符咒需要花費所煉法術精神力的二倍；而且有些符雖然可以製作，但卻無法施放於他人身上(如偵測邪惡術 detect evil )。另外要注意的是，在製作符咒時也是會失敗的。

　
　
前置技能 子午大法(time power)、鑑定術(identify)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座道士巨像聳立在這兒./tat statue
