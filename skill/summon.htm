 summon(召喚術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
召喚術 summon 40 30 0 0 67 0 0 60 10
　
簡介

　

召喚術，這個法術將會將你所要召喚的對象召喚至你現在的房間中，若是這召喚的對象在戰鬥中或是不和你同一個區域你都無法召喚他。


  • 此外：你在不可召喚的房間也不行。
    　　　他在安全房間也不行。
    　　　他在私人房間( PRIVATE ) 時也不行。
    　　　他在不可以 RECALL 的房間時也無效。

　
　
說明

　

冒險者系法術，很實用的輔助法術，施術者可以將目標從其他地點，瞬間帶來自己身邊，是一個再方便不過的法術了，也有非常多人，都會在很多地方擺一隻專門會召喚術( summon )的角色( 俗稱sum-char )，來幫助自己或他人到達想去的地方，可以省下很多的時間跟麻煩，也是剛入門的新手們，學大地之斬( earthslice )時最好的幫手。也可以用來將一些怪物直接帶到自己身邊，方便打一些物品，但若是目標等級比施術者高的話，是無法被召喚的。

　
　
前置技能 雲遊術(teleport)
相剋技能 法力加持(manamaster)
　
習得處 雕像館 - (白色聖光) 一座大冒險家巨像聳立在這兒./player statue
　 神秘高塔 - 召喚術掌門(Master summoner)正把守著通往另一端的通道。
