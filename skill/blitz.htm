 blitz(閃電奇襲式)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
閃電奇襲式 blitz 0 250 25 144 0 90 0 0 0
　
簡介

　

閃電奇襲式，學此技能後可以在戰鬥之中將自己的速度狂飆到極點而在同一回合內擊出多次連擊用以重創敵人。
　
　
說明

　

一般攻擊 (attack) 的最高階技能，可以於一回合內由一次攻擊增加到連揮四下攻擊，若是雙手都持有武器，則可以打出 4 × 2 = 8 下的連續攻擊，是四度空間內騎士最常用到的技能。並且因為使用一般攻擊不會花費任何的 hp / mana / move / power ，只會增加協調性，所以利用此技能，搭配上武器的隱藏法術，可以一邊攻擊，一邊獲得一些額外的法術支援。

　
　
前置技能 三段突刺(third attack)
相剋技能 劍訣(swordmaster)、獵殺(hunt)
　
習得處 雕像館 - (白色聖光) 一座騎士巨像聳立在這兒./knight statue
　 新奧夫卡 - (白色聖光) 奧夫卡女戰神黛安娜(Diana), 光榮的挺立在此.
