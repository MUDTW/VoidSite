 daub poison(抹毒術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
抹毒術 daub poison 0 60 0 0 0 0 0 0 12
　
簡介

　

抹毒術，可以將毒藥塗抹在手中的武器上，但失敗時自己會中毒。經過塗抹過的武器，在戰鬥時將可對敵人造成毒害，但離開遊戲後，其上的毒性將會自動消失。

　
　
說明

　

盜賊系技能，將毒藥施抹於武器上，造成攻擊敵人時，使之中毒的技能。具有時效性，且因為前置技能記憶點數需求高，造成配點吃緊，漸漸地已沒有人要學此技能了，加上現在有很多武器都有中毒的效果，機率比此技能還高，有興趣的人在可以玩玩看就好。

　
　
前置技能 背刺(backstab)
相剋技能 無
