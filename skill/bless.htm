 bless(女神庇祐術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
女神庇祐術 bless 3 60 0 0 36 0 0 50 8
　
簡介

　

女神庇祐術，此法術將會增加命中率，及一些攻擊性法術的抵抗力。

　
　
說明

　

牧師系技能，非常實用的技能，除了可以增加命中率之外，還可以增加降低被法術攻擊的威力，而且是聖光術( sanctuary )的前置技能，遊戲裡也有許多物品都可以上此法術。不管是那種職業都會用到他，近年來流行的抗魔裝，配上此技能，也才能達到完美的境界。

　
　
前置技能 無
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座牧師巨像聳立在這兒./cleric statue
