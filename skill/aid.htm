 aid(準確攻擊)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
準確攻擊 aid 0 300 25 0 0 45 27 0 0
　
簡介 　

準確攻擊。戰士系在正式格鬥中十分重要的技能，在戰鬥中，被你的對手躲避會使你的攻擊落空。擁有這項技能，可以降低被你的對手躲避的機會，可以說是十分實用的技能.

　
　
說明 　

經由多數玩家的實驗後，已經確定了命中率才是決定了是否能夠攻擊到怪物，而這個技能則是決定了「打到怪物後是否為 "絲毫不受影響" 的機率」，而不是學了此技能就能夠增加命中率因此準確攻擊( aid ) 算是一個十分雞肋的技能，因為學了之後絕對是弊大於利，還不如將空出的記憶點數用來學習其他法術或其他技能。目前四度空間中，玩家一般都是利用增加攻擊命中率的裝備來增加打中的次數畢竟大家想法的應該都是：先能夠打到Mob再說吧！因此這技能就見人見智囉。

　
　
前置技能 閃躲(dodge)
相剋技能 狂暴砍(heavyslice)
