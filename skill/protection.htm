 protection(聖祐術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
聖祐術 protection 34 30 0 0 54 0 0 50 8
　
簡介

　

聖佑術，將會使陣營為邪惡的生物對你所造成的傷害，減少一定的程度。

　
　
說明

　

牧師系技能，很實用的技能，能夠使玩家受到陣營邪惡的攻擊傷害降低約 20~25% 。此外，此技也是聖光術( sanctuary )的前置技能。遊戲中也是非常多能夠上此法術的道具，算是一個人人都需要的法術。

　
　
前置技能 女神庇祐術(bless)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座牧師巨像聳立在這兒./cleric statue
　 企鵝村 - (白色聖光) 聖潔般的神父(Father)正站在這兒聽你的懺悔。
