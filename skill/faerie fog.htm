 faerie fog(顯像術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
顯像術 faerie fog 73 20 0 0 18 0 0 20 8
　
簡介

　

顯像術，這個法術將會解除所有和你在同一個房間中人物的一些魔法影響。如：隱身術( invis )、藏匿( hidde )和潛行( sneak )。

　
　
說明

　

冒險者系法術，此法術可以將同一格房間內，上有隱身術( invis )、藏匿( hide )、潛行( sneak )效果的玩家或怪物，有機率率性地消除，如果沒有學偵測藏匿術( detect hidden )的玩家，可以用這招式去找出藏匿的敵人。

　
　
前置技能 偵測隱形術(detect invis)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座大冒險家巨像聳立在這兒./player statue
　 雕像館 - (白色聖光) 一座牧師巨像聳立在這兒./cleric statue
　 吉雅尼斯王城 - (白色聖光) 巫師的領袖梅傑特(mayato)正站在這.
