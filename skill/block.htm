 block(左手格擋)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
左手格擋 block 0 250 25 135 0 0 27 0 0
　
簡介

　

左手格擋，使用左手的盾來格擋敵人的攻擊。但是習有雙手持劍法的玩家在左手有武器時，也可以使用這個技能。這個技能在戰鬥中會自動使用。

　
　
說明

　

四度空間中屬於抵檔一般攻擊( attack )用的防禦技，目前因為有no save eq可以利用格擋時，放出法術攻擊敵人，所以漸漸成為另類的攻擊技能之一。很適合一些變形後的巫師和軍人。與格檔( parry )所需要的條件不同。

※比較

　 格擋(parry) 左手格擋(Block)
裝備 必為武器 武器或盾
發動條件 躲避能力高

約莫 -1300 以下

 躲避能力居中

約莫 -550~ -850
企圖心 皆差不多 越低則機率越高

由於此技能與格擋( parry )所需要的躲避能力值不同，所以技能配法需要取決於敏捷度和裝備重量(因為會影響躲避能力大小)。一般來說敏捷小於 26 的職業因為躲避能力較低，所以較適合配上格擋( block )。因為其敏捷低，自然躲避能力就會較低，洽好適合左手格擋的發動條件。如果敏捷要高又希望左手格擋的發動機率高，就得需要靠遊戲內會大幅減低躲避能力的裝備來達成效果。
