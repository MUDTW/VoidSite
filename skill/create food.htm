 create food(製造食物術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
製造食物術 create food 12 5 0 0 22 0 0 20 8
　
簡介

　

製造食物術, 這個法術會製造出名為 [string] 的食物。

　
　
說明

　

冒險者系法術，為新手練等必備的技能，作用為製造出乾糧，可以作為路標、羞辱人、塞滿怪物身上讓人不能解謎，但自從改有時效性後，當路標的功能雖然還在，但效果沒像以前一樣可以一人施恩，眾人受惠的功用了。但也因為改有時效，也可以避免怪物被塞滿，而不能給予解謎物品的情形。此技能因為有可以自由命名的功能，也常被用來當做留言給他人的功能。

　
　
前置技能 造水術(create water)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座冒險者巨像聳立在這兒./player statue
