 pick lock(開鎖)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
開鎖 pick lock 0 20 0 0 0 18 0 0 12
　
簡介

　

開鎖，透過練習可以撬開各式各樣的鎖，但是有些鎖是不能撬開的。

　
　
說明

　

盜賊系技能，能夠直接把一些需要鑰匙來開啟的門或物品打開，在某些程度上算是方便的技能。但在開鎖時，有機率失敗，並且如果有怪物在旁邊時，也會有機率被怪物所阻擋住，而不能開鎖的狀態發生。

　
　
前置技能 無
相剋技能 無
