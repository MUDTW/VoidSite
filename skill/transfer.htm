 transfer(傳送術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
傳送術 transfer 227 180 0 0 144 0 0 150 12
　
簡介

　

傳送術，可自由地來去於記憶中的房間，房間的記憶請查詢 HELP RNOTE ，一般施術者只能傳送自身前往目的地，唯有學過法力加持( manamaster )者才可傳送整個隊伍內的人員。

　
　
說明

　

冒險者系法術，非常實用的技能，可以將自己帶到一些特定的房間裡面，對於一些可能需要解謎或是受到很多敵人干擾，或是經過格數很多的地方來說，這技能可以替玩家省下非常非常多的時間。此外，如果玩家是屬於牧師系，學有法力加持( manamaster )，就可以一次傳送多位玩家到指定區域，這也是牧師系最常使用到的功能之一。

【註】精神力若是過低，則無法集體傳送隊伍中的人員。

　
　
前置技能 雲遊術(teleport)
相剋技能 無
　
習得處

　

 風采之都 - (白色聖光) 笑臉迎人的打工巫師(wizard)以其無窮盡的【瑪那】努力中。

【註】需支付 6000000 元
　 魔宮 - 笑容可掬的傳送神(god)正詢問你準備要好出發了嗎!
