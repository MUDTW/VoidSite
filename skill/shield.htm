 shield(氣盾術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
氣盾 shield 67 30 0 0 40 0 0 40 8
　
簡介

　

氣盾，這個法術將會增加施法者的防護等級( ac )。

　
　
說明

　

巫師系技能，法術效果為增加防護等級( ac ) 60 點，屬於中階的輔助法術，可以利用此法術將防護等級撐到 1000 以上，來變成特攻打不痛的肉盾。

　
　
前置技能 神靈呼喚術(armor)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座力場系巫師巨像聳立在這兒./wizard statue
　 新奧夫卡 - (白色聖光) 奧夫卡的女祭司(Priestess)正在這以她誠摯的笑容為你祝福.
