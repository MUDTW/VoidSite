 chaodu(超渡)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
超渡 chaodu 0 80 0 0 108 0 0 100 12
　
簡介

　

超渡，犧牲自己的內力及法力，可將屍體傳送回地藏廟。但失敗時屍體有損傷之虞，使用時務須小心。

　
　
說明

　

道士系特殊技能，此技能可以將沒練生命水晶而死去玩家的屍體，送到位於 /3sw 的地藏廟。在現在已經慢慢的不實用了，因為不帶生命水晶去亂闖，本來就是很危險的一件事，如果死去後未撿屍體時，系統一但 Crash ，那麼玩家的裝備、金錢都將會消失，而且大神是不會賠償的，所以目前用到機會是越來越少了。

　
　
前置技能 操屍術(charmcorpse)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座道士巨像聳立在這兒./tat statue
　 長安城 - 一位身穿黃色道袍的茅山道士正揮舞著桃木劍在做法./taoist
