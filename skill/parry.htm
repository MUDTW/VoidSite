 parry(格擋)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
格擋 parry 0 200 25 117 0 22 27 0 0
　
簡介

　

格擋，學得此技能後，將可在戰鬥之中以武器化解掉敵人的攻擊。

　
　
說明

　

騎士系重要防禦技能，是一種適合高敏捷且躲避能力( dc )高的防禦技。主要是利用右手所持的武器來化解敵人的一般攻擊( attack )。與左手格檔( block )所需要的條件不同。

※比較

　 格擋(parry) 左手格擋(Block)
裝備 必為武器 武器或盾
發動條件 躲避能力高

約莫 -1300 以下

 躲避能力居中

約莫 -550~ -850
企圖心 皆差不多 越低則機率越高

由於此技能與左手格擋( block )所需要的躲避能力值不同，所以技能配法需要取決於敏捷度和裝備重量(因為會影響躲避能力大小)。一般來說敏捷 28~30 的職業因為躲避能力較高，所以得配上格擋( parry )。但躲避能力值並不是越高越好，因為高到某一個程度後，會慢慢的被閃躲( dodge )所取代。所此如果要靠此技能做防禦的玩家，請謹慎面對配裝備的問題，如果裝備配好後，躲避能力偏高，則可以考慮換上其他類型的裝備，一樣可以使格擋( parry )以較高的機率出現，但
可以更強化自身職業的能力。

　
　
前置技能 閃躲(dodge)
相剋技能 無
