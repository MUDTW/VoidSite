 charmcorpse(操屍術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
操屍術 charmcorpse 0 50 0 0 108 0 0 100 12
　
簡介

　

操屍術，相傳在中國湘西一帶，道士可以利用法力，驅使屍體聽其號令。而此技能便是使屍體變成殭屍，而聽令於你。

　
　
說明

　

道士系特殊技能，與攝心術( charm person )性質很接近的一種技能，但操縱的對象為已死去的怪物屍體，由其轉化成殭屍( corpseman )，有時間的限制，但是因為殭屍擁有不死之身，會一直爬起來繼續攻擊，所以常常用來擋一些比較大型的怪物，加上如果給這種不死之身的殭屍雙手拿上一些法術性武器，會讓他的攻擊威力變超高。等級越高的屍體，變成的殭屍等級就越高，攻擊力也越強，會玩的就知道利用多隻殭屍會等於多了好幾個幫手，而且是不怕死的幫手！

※特性

  • 殭屍的有效時間為 PM 3點 ~ AM 10點。
    在非上述的時間將無法招喚，而已招喚出的殭屍將連同你給予他的物品一同消失。
  • 可利用十字光芒( cross glory )來「殺死」殭屍。
  • 失去控制的殭屍，無法再用攝心術恢復其控制。

　
　
前置技能 攝心術(charm person)、子午大法(time power)
相剋技能 無
