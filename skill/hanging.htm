 hanging(絞殺)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
絞殺 hanging 0 300 40 162 0 72 0 0 20
　
簡介

　

絞殺，在叢林野戰時，士兵中常常會躲在暗處，乘敵人不注意之際，給予敵人致命的一擊。這就是可怕的絞殺，也是軍人必受的基本訓練之一，通常乘敵人『不備』之時，使用出來會有更不同的效果。

　
　
說明

　

軍人系技能中，看攻擊傷害率( dr )決定威力的技能。是一種不會花費hp mana move power的攻擊，但是此技能威力的浮動值很大，如果最高攻擊算 100%，那攻擊威力會在 60%~100% 之間跑，但整體來說攻擊算是滿強的。
　

※特色和限制：

  • 使用時右手不可以持有任何裝備。
  • 攻擊目標若是處於睡眠狀態下，技能威力將增加約 20%~30%。
  • 攻擊目標若是處於看不到的狀態下，這裡又分為二種狀況
    A.如果自身有隱身術( invis )或藏匿( hide )，且攻擊目標看不到，則威力約增強 10%~30%，且只會計算其中一個，沒辦法累加威力，即隱身術( invis)或藏匿( hide )皆有時，只計算其中一個。
    B.攻擊目標若是屬於「瞎」的狀態下，則威力增加約 20%~50%。
  • 攻擊目標若是處於戰鬥中則無法絞殺，也就是攻擊目標一定要在被動狀態下，而絞殺一定是第一個攻擊技能。
  • 絞殺威力可以上述第二及第三點來得到最大威力。

　

【註】以上增加威力的數據，請有更準確數據的玩家來幫忙確認。
