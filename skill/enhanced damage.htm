 enhanced damage(加強破壞)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
加強破壞 enhanced damage 0 100 25 27 0 0 27 0 0
　
簡介

　

加強破壞, 學會此技巧之後, 在戰鬥中將更能掌握武器的使用方法而造成更大的傷害力。

　
　
說明 　

玩家在使用一般攻擊( attack )時，若有學習此技能，則攻擊威力將增加約莫 2/3 倍，並且是少數幾個非拳法家士、劍士系技能，卻又有加內力上限值的騎士系技能。因此許多玩家在練所謂遊俠或是大拳頭騎士等職業時，都會學習此技能用以增加攻擊力。

　
　
前置技能 無
相剋技能 無
