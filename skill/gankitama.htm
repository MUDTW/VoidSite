 gankitama(元氣玉)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
元氣玉 gankitama 0 400 60 90 0 0 180 0 8
　
簡介

　

元氣玉，可將天地之間所有生命力聚集在自己的身上，而向敵人發出致命的一擊，但其威力將視個人之內力修為而有所不同。

　
　
說明

　

拳法家系高階拳技，屬於界王拳系的第二招，由著名漫畫作家鳥山明的成名作品《七龍珠》裡取材而出。施展時會消耗內力值上限的四分之三，可以打出很高的攻擊，但很不實用，因為施展完此招就幾乎沒有剩餘的內力了，且威力只略高於降龍十八掌( dragonfist )。另外，此招練習過程十分艱苦，想要練隻界王拳系拳法家的玩家要有心理準備。

　
　
前置技能 龜派氣功(kamekameha)
相剋技能 無
