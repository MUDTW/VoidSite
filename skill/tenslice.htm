 tenslice(十字斬)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
十字斬 tenslice 0 300 75 315 0 270 0 0 50
　
簡介 　

十字斬， 為亞瑟王想出以對抗薩拔的絕招。在第一次集氣之後，第二次以快速的劈砍，對敵人駛出兩道攻擊。也由於需要耗費大量的體力，故使用後會損耗血量及移動力。

　
　
說明

　

騎士系高階技能，擁有固定的破壞能力，並且是範圍攻擊。每次出招會消耗掉生命力 200以及移動力 300，且現有生命力需超過 400 ，移動力必須超過 800 才可使用，對騎士系來說是一個非常好用的瞬間高攻擊技。因為有些敵人特性就是非常會逃跑，而騎士的其他主要技能都是以多段攻擊為主，往往才打到第一下，對方就跑掉了，因此追殺起來很累。但如果有了這招可以給予較多的傷害，也比較省時間，此外這招在對玩家施放時，會變成攻擊二次，但對mob施放時會變成單體技。施放時也要特別注意，因為此招增加的協調性不低，以及範圍技的特性，如果時機拿捏不好，很容易變成自殺技。

【註】此技能施放時沒有delay time

　
　
前置技能 雙手持劍法(two weapon)、風嘯之斬(tenslice)
相剋技能 無
　
習得處

　

 騎士堡 - (白色聖光) 一位傳說中偉大的騎士亞瑟王(Arthur)站在你的面前。

【註】需解完謎題 - 擊殺薩拔。
