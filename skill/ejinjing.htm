 ejinjing(易筋經)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
易筋經 ejinjing 0 200 0 0 0 0 360 0 0
　
簡介

　

易筋經，傳說中少林寺失傳己久的高深內功，學習此內功的呼吸吐納之法，可以令人脫胎換骨， 加速生命力的恢復。

　
　
說明

　

拳法系加快恢復血量、內力的技能，可以縮短恢復時間，也算是拳法家的必學技能。與另一個恢復的技能葵花寶典( okra power )不同的是，後者恢復的量較少，但能加強辟邪劍法( evilslash )的威力，而前者恢復的量較多，卻沒有能讓招式強化的功用。

　
　
前置技能 無
相剋技能 葵花寶典(okra power)、辟邪劍法(evilslash)
　
習得處 雕像館 - (白色聖光) 一座拳法家巨像聳立在這兒./bxr statue
