 pretend(偽裝)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
偽裝 pretend 0 50 0 0 0 0 0 0 12
　
簡介

　

偽裝，當你使用偽裝術之後，別人將無法分辨你和被你偽裝的對象，可以用來欺敵。晚上使用時，可以防止主動攻擊的敵人對你進行攻擊，但是一旦進入戰鬥，就有機會被敵方識破。

　
　
說明

　

盜賊系技能， 可以讓施術者偽裝成指定目標相同，使其他玩家誤以為是你所偽裝的怪物在面前的技能，並且在室外天黑時，搭配藏匿( hide )、潛行( sneak )、隱身術( invis )，也會使一些怪物攻擊你的機率大大地降低。此外，在偽裝的狀態下，別的玩家輸入 where 指令是可以看到此區域的全體玩家的(包括處於偽裝的玩家)，但 where [victim] 時，指定目標如果是偽裝狀態，則只會出現他偽裝的怪物名稱。被攻擊的時候，機率性會使得偽裝被破解而恢復原來自身的敘述，再輸入一次 pretend 也可以停止繼續偽裝對象。簡單來說，此技能就是欺敵專用。

　
　
前置技能 藏匿(hide)
相剋技能 無
