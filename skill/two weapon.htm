 two weapon(雙手持劍法)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
雙手持劍法 two weapon 0 600 25 90 0 0 0 0 0
　
簡介

　

雙手持劍法，這個技能可以使得你具有於左右手都裝備武器的能力。並且在一回合之內，左右手的武器都可以傷害到敵人。

　
　
說明

　

騎士系重要技能。在未學習此技能時，左手只能夠持盾牌類裝備，並且使用一般攻擊( attack )時也只有右手能攻擊的到敵人，但是於學習此技能後，左手不但也能夠持有武器，並且使用一般攻擊時也會有左手的攻擊，如果在習有多段攻擊( second attack / third attack )類技能的狀況下，此時左手也能夠具有多段攻擊的能力。此外，一些本來前置技能為雙手持劍法的技能，在習得後將左手武器脫掉的狀況下，也可以使用。但如果是屬於多段攻擊的技能，則左手的多段攻擊會沒辦法出現。(如 : 風嘯之斬 windslice)

※特性：

  • 左手只能夠裝備小於 21kg 的武器，超過 21kg 會拿不動。
  • 左手的攻擊會以右手拿的武器類別為主。因此如果左手拿的是非正常範疇類的武器(如：槍炮、弓件、手杖類)，左手會只顯示揮拳，但實際上的效果或隱藏法術還是有效的。
  • 左手的攻擊力只會有右手約莫 60~70%的威力(武器攻傷值相同下)。因此若是左手的武器攻傷值越高，對左手的攻擊力越有幫助。
  • 左手拿盾牌類後就沒辦法造成傷害。
  • 左手拿武器，使用非一般攻擊技能不會出現二次攻擊，鎗也不能射二次。
  • 右手不拿武器也不會出現左手的攻擊力。
  • 左手所持武器不會被擊落。

【註】有些劍類武器也是只會出現揮拳的動作，而不是持有武器砍人的敘述。請想像成一些比較特殊，需要握拳動作的武器就可以了。
