 kamekameha(龜派氣功)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
龜派氣功 kamekameha 0 300 60 45 0 0 135 0 8
　
簡介

　

龜派氣功，為龜仙人 - 武天老師的得意絕技之一，發功時將全身之氣集於雙掌之中，再一股作氣向敵人打去，內力強度愈強則威力愈大。

　
　
說明

　

拳法家系高階拳技，屬於界王拳系的第一招，由著名漫畫作家鳥山明的成名作品《七龍珠》裡取材而出。施展時會消耗內力值上限的四分之一，但所打出來的威力並不高。此技能跟元氣玉一樣難練，練功方法也差不多。

　
　
前置技能 拳力(bare fist)、易筋經(ejinjing)
相剋技能 劍訣(swordmaster)
