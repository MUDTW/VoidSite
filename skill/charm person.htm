 charm person(攝心術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
攝心術 charm person 7 80 25 0 108 0 0 120 8
　
簡介

　

攝心術，這個法術如果成功可以使得被施法的人會聽你的話。你可以利用指令 Order 來命令他作一些事。你必須對被你所馴服的人的行為負責，相反的如果別人砍了被你馴服的人，他一樣要接受懲罰。

　
　
說明

　

巫師系法術，超級實用的技能，可以利用這技能練功、升級、打裝備、 pk 、整人、惡搞，由於太過黑暗層面，這技能請勿亂用，想體驗的人請自己練一隻......在這也不說那些太黑暗的用法......。而被施法者，被主人用某些技能攻擊不會反擊，這些也請自己去試......但如果被反擊，此技能效果將會自行解除......強力建議一定要練一隻會此技能的角色，這樣可以方便做太多事情了。

※特性

  • 施法者本身智力( int) 必須大於或等於被施法者。
  • 施法者本身等級必須大於或等於被施法者。
  • 有些特定的怪物不受此法術影響。
  • 可利用 order 指令使被施法者做某些事。
  • 若是被施法怪物為 55 級以上，則可以利用組隊指令 ( group )，搭配牧師系角色替怪物補血，然後可以用來升級，方便練等。
