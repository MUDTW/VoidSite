 gunner(鎗法)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
鎗法 gunner 0 350 0 90 0 90 0 0 0
　
簡介

　

鎗法，為使用槍砲、弓箭射擊者之無上心法，鎗法熟練者將可發揮鎗枝的最大威力，且可百步穿揚，直接命中敵人要害。

　
　
說明

　

射擊的加強威力技能，有了此技能後，使用射擊( shoot )時才會有足夠威力，和葵花寶典( okra power )及加強背刺( bs power )的作用功效是類似的意思，都是為了使某一個技能，能夠打出高數字用的一種特殊技能。

　
　
前置技能 射擊(shoot)
相剋技能 無
