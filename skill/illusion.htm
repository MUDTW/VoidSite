 illusion(迷朦幻影)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
迷朦幻影 illusion 242 50 0 0 63 0 0 50 10
　
簡介

　

迷朦幻影，此一法術將可使其它人在看施法者時產生幻像，而無法正確分辨施法者的位置。

　
　
說明

　

巫師系法術，效果是機率性的讓對方的攻擊無效一次，若是施術者的智力值高於對手越多，則機率越大，配合上目盲術( blindness )來打一些大型怪物，也可以十分輕鬆地將怪物解決。是一個非常值得一學的技能。如果想打一些中高階的裝備，就可以練就一隻會此技能的職業去打，可以省下許多的麻煩。

【註】可搭配魔法沙漏( magic clock )來降低敵人的智力值，使此技能的機率大幅提昇。

　
　
前置技能 偵測隱形術(detect invis)、全隱術(mass invis)
相剋技能 無
　
習得處 神秘高塔 - 一隻混合著七種顏色的三頭龍(Illusionist)對你嘶嘶的叫著.
