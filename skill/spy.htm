 sneak(偵查)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
偵查 spy 0 50 0 0 0 45 0 0 10
　
簡介

　

偵查，這個技能可以用來偵查你四周的玩家或者是怪物。和 scan 不同的是，其有效距離長達四格房間，可以更清楚的了解四周的環境。

　
　
說明

　

盜賊系技能，為搜查( scan )的加強版，可以查看以自身為中心，往外延伸出去的四格房間。在區域較大，且有很會亂跑的怪物時，是一個非常適合的技能，用在一些有迴圈迷宮上也很方便。
