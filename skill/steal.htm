 steal(偷竊)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
偷竊 steal 0 30 25 0 0 63 0 0 12
　
簡介

　

偷竊，讓你能在別人不注意時，竊取他身上的東西或是金錢，不過如果失敗了也必須付出相當的代價。

　
　
說明

　

盜賊系技能，可以竊取等級小於或等於自身的目標身上物品和金錢，在某些時候非常方便，但偷竊失敗時會被喊出「 ○○○真是個不要臉的賊!!'. 」，而且會加上一個(小偷)的字樣。而有著此字樣的人，玩家便可以在一些保護區追殺你，此效果也可提供給他人練功之用。

【註】擁有(小偷)光環的人，無法主動攻擊玩家，只能被打，並且會被許多怪物厭惡，將會不能購買物品，且少數特定怪物見到你就打也會。

　
　
前置技能 窺視(peek)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座小偷巨像聳立在這兒./thief statue
