 heavyslice(狂暴砍)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
狂暴砍 heavyslice 0 250 25 90 0 0 27 0 0
　
簡介 　

狂暴砍，在戰鬥中，你的對手隨時可能會以武器或是盾牌來阻擋你的攻擊，但是如果你學會了這項技能。此技能將會使你的對手，因為承受不了你巨大的力量，而失去了抵擋的能力。

　
　
說明

　

騎士系技能中的一個雞肋技能，此技雖然可破除左手格擋( block )的技能效果，為一主動技能，但因學習此技後卻不能學習雙手持劍法( two weapon )，使攻擊威力大減，致使沒有理想的攻擊招式，推薦玩玩就好，不必太鑽研。

　
　
前置技能 左手格擋(block)、加強破壞(enhanced damage)
相剋技能 準確攻擊(aid)、雙手持劍法(two weapon)
　
習得處 雕像館 - (白色聖光) 一座騎士巨像聳立在這兒./knight statue
　 騎士堡 - (白色聖光) 一位傳說中偉大的騎士亞瑟王(Arthur)站在你的面前。
　 Ultima - 戰士傑佛瑞(Geoffrey)正帶領著他的軍團經過你的眼前.
