 barter(五鬼運財術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
五鬼運財術 barter 0 30 0 0 45 0 0 0 0
　
簡介 　

五鬼運財術，古代道士的專有祕法，可藉由咒語將屍體內剩餘的靈力轉換成金錢。

　
　
說明

　

道士系特殊技能，是早期玩家用來賺錢的技能，利用sacrifice corpse時，視屍體等級來決定金錢的多寡，早期因為可以快速且大量地領取炸彈( 於小精靈村 )，可以一整晚靠炸彈屍體賺錢，收入相當地可觀。但經神族修改後，目前如果要靠這招賺錢，建議還是打怪比較實在。

　
　
前置技能 子午大法(time power)
相剋技能 無
