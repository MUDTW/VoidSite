 dispose(佈陣)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
佈陣 dispose 0 320 0 150 120 200 0 0 15
　
簡介 　

佈陣，為軍師之基本技能，其相關的陣法一共有十二種，各有不同的影響，為一群體行動時的良好助力。習有此技能者鍵入 dispose可以有較詳盡的說明喔。

　
　
說明

　

說到這個技能，就必須跟一些陣法一起說明一下，會學到這個技能，就代表是練到軍師這個職業,而軍師是使用陣法的職業，但是各種陣法非常難練，而且成功率非常低，加上成功的時候才會增加技能熟練度，實用度也低，目前只有一個少許陣法稱得上好用，建議一定得對遊戲有相當程度的瞭解之後才來試試看這職業及技能。

※特性

  • 佈陣時消耗精神力及移動力，不論成功與否都會減少。
  • 佈陣者一定得是隊長(若無組隊狀態下，則隊長就是本身)。
  • 佈陣後若有隊員脫離隊伍，則陣型解散。
  • 佈陣可於戰鬥中施展。

　
　
前置技能 無
相剋技能 法力加持(spellmaster)、魔力加持(manamaster)、元氣玉(gankitama)
　
習得處 雕像館 - (白色聖光) 一位穿著奇特的謀略家(ploter)勤勞的講解各式陣法的奧義
