 scan(搜查)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
搜查 scan 0 10 0 0 0 0 0 0 8
　
簡介

　

搜查，這個冒險者常用的技能，可以用來查探四周的敵人，但僅限於一格房間的距離。建議在危險的區域，在移動前，先用 scan 來調查四周的情況。

　
　
說明

　

一個很實用的技能，可以查探與自身相連的其他房間有什麼怪物及玩家，以免玩家走過去後被眾多怪物圍毆，此外還可以用此招來查看一些路標或是想要找尋的特定目標。也是走迷宮型區域的好幫手。

　
　
前置技能 無
