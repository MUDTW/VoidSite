 disarm(卸除武器)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
卸除武器 disarm 0 200 25 0 0 0 27 0 8
　
簡介

　

卸除武器，乃是一項可將敵人手上的武器擊落的戰鬥技巧，在使用此技能時，右手必須裝備著武器。

　
　
說明

　

一個非常實用的技能，可以讓拿著武器的敵人，強制卸除掉右手的武器。但遇到習有劍訣( swordmaster )的人，就比較難卸除他的武器。此外如果是mob被卸除武器後，武器還是會在mob身上；但如果是玩家被卸除武器，武器將會掉落至地面(脫離玩家身上)，這點要特別注意。此技能是依賴武器生存職業的剋星。一般練功都是在pk場讓玩家互相練功(速度快)，或是找會穿上武器的mob練功(速度慢)。要練此技能，請盡量挑比較沒人在的時間練，因為pk場還有其他用途的。

【註】一定要右手持有武器才能夠使用，左手武器沒有辦法卸除掉他人武器
