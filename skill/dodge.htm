 dodge(閃躲)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
閃躲 dodge 0 100 25 18 0 45 0 0 0
　
簡介

　

閃躲，此技能將會讓你閃躲掉對方的攻擊，使你不受任何的一般傷害。

　
　
說明

　

實用的防禦技能之一，可以閃避掉絕大部份的一般攻擊( attack )，且機率很高。閃躲機率與輕功( agility )相同，是以躲避能力( dc )來決定閃躲機率，躲避能力越高則閃躲機率越高；另外企圖心( aggressive )的高低也會影響，一般來說企圖心越低，閃躲機率越高。

【註】攻擊企圖心若越高，則偏重攻擊，因此防禦這方面的機率就會降低。

　
　
前置技能 無
