 manamaster(法力加持)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
法力加持 manamaster 0 850 0 0 270 0 0 0 0
　
簡介

　

法力加持，藉由宗教上信仰神的力量，將本身的法力提昇，天神也會因為信仰者的忠誠而在奉獻時提昇奉獻者之精神力。

　
　
說明

　

牧師系技能，為牧師的重點技能，學習此技能後，可以將許多法術的特性優化，也是許多牧師系範圍法術的前置技能，一般學到此技能後才有資格稱為純牧師。

※特性：

  • 習有此技能者，精神力恢復速度將增加一倍。
  • 習有此技能者，在 sacrifice corpse 時可以機率性回復精神力。
  • 習有此技能者，使用傳送術( transfer )時，可以將自己及同一房間內的隊友一併傳送。

　
　
前置技能 聖光(sanctuary)、完全治療術(full heal)
相剋技能 魔力加持(spellmaster)、子午大法(time power)
　
習得處 雕像館 - (白色聖光) 一座牧師巨像聳立在這兒./cleric statue
　 吉雅尼斯王城 - (白色聖光) 法力高強的梅林(malin)正站在這堨H他高超的法術為所有人治療.
