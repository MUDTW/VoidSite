 shoot(射擊)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
射擊 shoot 0 600 25 90 0 0 0 0 11
　
簡介

　

射擊，當使用射擊這個技能的時候，你的右手必須拿著射擊用的武器，而且武器必須上載著彈藥或是箭枝。射擊和你自身的攻擊傷害率( dr )無關。強度是決定在射擊武器的威力，這是軍人強大的技能之一。當槍枝發生膛炸而壞掉時，可以到 /wwn 找蝙蝠俠( batman )修理(見 fixeq )。

參考：<LOADMISSILE> <UNLOADMISSILE> <FIXEQ>
　

　
　
說明

　

軍人系射擊技能，也是目前最夯的主流技能之一。需要搭配鎗法( gunner )才可以得到高殺傷威力。也是目前獨立性最高的技能，因為所需要的記憶點數較低，可以搭配其他職業的各種技能跟法術，也是目前唯一可以雙修除了拳法家外各種職業的技能。遊戲中更是有各種超強破壞力的槍械，也造成了目前此技能不管是獨立逛區域，或是組隊打大型mob，都非常方便及暴力的技能。但因為要使射擊出來有一定威力，所穿裝備的也要有一定的水準，所以不建議新手一開始就練，但有招一日一定要練一隻適合自己的軍人來試試。

特性：

  • 習有鎗法( gunner )，威力可增加約 300%。
  • 射擊要有威力要看攻擊命中率( hr )以及睿智( wis )是否足夠。
  • 攻擊命中率( hr )方面：所需的 hr = 40 + (str × 2)；其中每增加力量( str )一點，可以增加兩點的攻擊命中率
  • 睿智( wis )方面：睿智的最低需求是 20點，再往上增加的話，若為單數，則單發槍械的威力會增幅一些；若為偶數，則增加連發槍械的威力。
  • 如果因中了會降低力量( str )的法術(如寒霜之處 chill touch)，則所需的攻擊命中率( hr )，也會因此下降。
  • 練功請用連發槍械，可以快速增加熟練度。
