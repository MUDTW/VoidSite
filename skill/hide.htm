 hide(藏匿)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
藏匿 hide 0 250 0 90 0 90 0 0 8
　
簡介

　

藏匿，使你行蹤能夠不被他人知道。

　
　
說明

　

盜賊系技能，可以使玩家或敵人看不見自己，且效果較隱身術( invis )更佳，因為目前的怪物或是玩家比較少擁有偵測藏匿術( detect hidden )。有些技能也會因為目標看不到自己而有加強威力的效果( 如背刺 backstab、絞殺 hanging)，所以這技能也是有幫某些技能加分的效果。

　
　
前置技能 偷竊(steal)
相剋技能 無
　
習得處 雕像館 - (白色聖光) 一座小偷巨像聳立在這兒./thief statue
