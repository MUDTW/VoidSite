 abscond(遁逃術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
遁逃術 abscond 0 120 20 45 0 135 0 0 8
　
簡介

　

遁逃術，為東瀛忍術，可於戰鬥中自由決定所要脫逃的方向，但是所有不可進入的地方依然不可進入，屬於強力的脫逃術。

　
　
說明

　

盜賊系逃脫技能，特殊點是在即使協調性破千依舊可以使用，也可以說打大型怪物所用的去除魔法術( dispel magic )的搭配技能。利用這技能可以指定逃脫方向，也可以在逃脫後不被怪物所繼續攻擊，但對於一些擁有 holylight 效果，又是主動攻擊的怪來說，是他的剋星。

【註】Holylight是Imm及怪物的特殊指令，開啟後的效果為不論目盲、刺瞎與否，必定看的見。

　
　
前置技能 閃躲(dodge)、背刺(backstab)
相剋技能 無
