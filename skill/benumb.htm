 benumb(迷香術)
　
 　
Skill Name Slo Mem Cyc Hp Mana Move Power Cost Beat
迷香術 benumb 0 120 0 0 0 25 0 0 10
　
簡介

　

迷香術，施術者可以向隔壁房間施放迷香，使該房間內怪物或玩家進入睡眠狀態。但失敗時會波及自己所在的房間。

　
　
說明

　

盜賊系技能，非常實用的一個技能，可以使指定房間內的怪物處於睡眠狀態，而且成功率高，又可以不必與
怪物在同一房間內就可以使用。當一個區域經過迷香施放過後，可以減少玩家的死亡機率跟解謎所需時間，因此是很多人在打至尊級怪物時，前置作業的第一步。目前更流行到連解謎區域都常看到有人去玩，算是很另類但是一直很流行的技能。

※特性

  • 迷香術只對比小於施放者自身等級 +10 以內的怪物有效，所以玩家只能迷倒等級 70 以下的怪物。
  • 迷香術只對智力( int )小於等於施術者的怪物有效，所以一般此類職業智力都會練 30。
  • 迷香術施放失敗時，迷香會飄回施放者房間，迷倒自己。
  • 若於非戰鬥區域施放時，除非玩家有(小偷)光環，不然無法迷倒玩家。
  • 若是遇到智力( int )比施放者高的怪物，可以利用困惑術( confuse )或是魔法沙漏( magic clock )等法術來降低怪物的智力。
  • 迷香術所需物品，可至區域 SOGO 百貨 購買氰化鉀/chemical poison。

　
　
前置技能 背刺(backstab)
相剋技能 無
