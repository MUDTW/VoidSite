○

 MUD 定義
　
 　

　　MUD (Multiple User Dimension 多人世界、Multiple User Dungeon 多人地下城、Multiple User Dialogue 多人對話) 俗稱『網路泥巴』遊戲；是種使用者能進去冒險的電腦程式。每個人控制著一個電腦化的角色/實體/化身。你可以四處逛、和別人控制的角色談天、在有怪物滋擾的危險地方探險、解決謎題、甚至創造你私人擁有的房間；敘述和物品。

　　MUD 可以實現在現實中根本無法體驗到的人生、感受，在 MUD 中你可以是一名威風凜凜而又樂於助人的大俠，也許是一個殺人不眨眼的大魔頭，這都全憑玩家高興，只要你喜歡，你可以扮演任何不同的角色或是擁有多種以上的性格。

　　MUD 是個虛擬世界，所有角色和事物都只是玩家或設計者意願的體現，既真實又虛幻，正是這種感覺傾倒了世界各地的無數玩家，每天都有來自世界各地的上千萬玩家在分佈於世界各地的 MUD 站上奮鬥，營造自己的夢想。


  
○ MUD 歷史
　 　

　　第一個 MUD 是由 Roy Trubshaw 和 Richard Bartle 於1979年在 Essex 大學所寫的『MUD1』。最初的研發動機只是為了測試共用記憶體系統 (shared memory system) 而不是為了遊戲，遊戲的構想是之後才慢慢發展出來的。MUD1是　　VAX/VMS 主機上運行，最初可讓玩家在虛擬的地方中走動，之後的版本才加入了物品、指令…等。

現代 MUD 的發展源於1989年由一位 CMU 大學生 － Jim Aspnes 於週末時所寫的遊戲程式，命名為 TinyMUD。由於它具有良好的可移植性，很快便廣為流傳而發展至今。在TinyMUD 之後，許多程式設計員又開發出多種 MUD 的程式碼，形成多種流派，較有影響的如 DikuMUD、LPMUD、MOO等。

簡單來說，這個網路上的虛擬世界是由文字所建立的，對於以前電腦配備及網路頻寬而言，是可滿足所有電腦玩家的多人網路角色扮演遊戲。隨著電腦的多媒體發展及網路寬頻的技術，很多電腦遊戲 (PC Game) 已拓展為網路遊戲 (On-line Game)，甚至國內也相繼出現了圖形化的 GMUD (Graphic MUD)，對於純文字 MUD 當然帶有一定的衝擊，但是卻未影響它的生存空間，因為所有純文字 MUD 是 24 小時的免費服務，而且文字的幻想空間是非常遼闊的，不受圖形及音效先入為主所約束。

大部份國內的 MUD 管理者都是由學生所組成的，而且 MUD 的系統也常放於大學的工作站(Workstation) 上，就因為學生的無條件付出及搭上學術網路的便車，使得 MUD 這個非謀利的服務得以在國內生存及蓬勃發展。

　
○  MUD 種類
　 　

基本上 MUD 可分為三大種類：

　　◆ 戰鬥導向的 MUD
　　◆ 社會導向的 MUD
　　◆ 其他

　　戰鬥導向就是說戰鬥是該 MUD 所固有的文化。在裡面的玩家都半是以合作或是獨自殺怪獸、解決謎題、和一起探險而從中獲取經驗值、金錢或裝備。

　　而一個社會導向的 MUD 就完全不同了，你可能在扮演一個角色，進行社交的活動 (當然也可能包含戰鬥)，又或許只是純粹在與朋友們聚會、聊天、開開玩笑、或是討論各種事物。

　　屬於其他類別的 MUD，大部份都不是以戰鬥為主的，可能是用作遙距教學之用，或是商業性的網路會議。要強調的是 MUD 的形式都是很有彈性的，是根據管理者的意念而可各有不同，因此戰鬥導向可以包含社會結構，而社會導向也可以俱備戰鬥探險，簡單來說，你可以把 MUD 視為一個現實或虛幻的社會縮影，又或者是個商業或教育性的虛擬世界。
　
○ MUD 好玩性
　 　　

　　MUD 跟電腦遊戲 (PC Game) 最主要的不同之處，就是它沒有 GAME OVER，而且它給予每個玩家極大的自由度，你可以隨個人意志自由行動，而不必跟著早已設計好的劇情呆板的玩下去 (現在雖然有許多 PC Game 設計出所謂的 "多線式" 劇情，但是其自由度仍比MUD 略遜一籌)。

　　更重要的是，在 MUD 中與你結伴組團去冒險、打怪物的不再是系統創造出的虛擬人物 (Virtual Character)，而是與你同樣坐在電腦前的玩家。因此 MUD 的好玩之處，在於遊戲中人與人之間的互動。「MUD就像一個小型的社會縮影」，在新手成為老手的過程中，你必須不斷的向別人請教，有時你也必須跟別人組團 (group) 共同去打怪物；以獲取裝備或金錢。藉此你可以在遊戲中認識許多新朋友，拓展自己的人際關係。
　
○ MUD 遊戲概念
　 　

　　純文字 MUD 是不佔玩家半點電腦硬碟空間的，而且不要求使用者電腦配備，只要使用者擁有連接網路 (Internet) 的能力，即可享受 MUD 帶來的樂趣，也正因如此的方便，就算玩家擁有的是一台古董 286 電腦，也不會影響遊戲的效率。


　　MUD 是由文字所組成的，即是一切的訊息、環境、事物、人物、動作、活動等都是以文字方式表達，可能對於一個剛接觸 MUD 的人而言，剛開始時會有點不習慣，而且可能對於螢幕上的快速捲動訊息產生手忙腳亂的情形，這些都是適應問題，假以時日你便會控制自如。

　　遊戲的人與人之間溝通是用打字形式去表達的，你可想像如網路聊天 (IRC、ICQ等) 一樣，而玩家的一切動作都是通過下達指令而完成。玩家所存在的地方單位，我們簡稱為房間(Room)，由眾多房間組成的便是我們俗稱的區域 (Area)，玩家通過下達移動指令 (如：東(e)、南(s)、西(w)、北(n)、上(u)、下(d) 等) 可通往另一房間於區域內探險，區域可以想像成故事或場景，一個 MUD 可以由無窮無盡的故事或場景所結合而成，而且優點
在於 MUD 的故事或場景的增加或刪除是非常有彈性的，因此造成 MUD 的耐玩性比 PC Game 為高。

　　遊戲中的怪物 (NPC) 是跟著設計者的程式去動作的，憑著無限的幻想力可創造出活龍活現的人物，通過程式的開發，授予 NPC 的人工智慧，結合房間的描述，一個個栩栩如生的精彩故事便浮現於眼前。

　　總括一句「MUD 遊戲是創造夢想的樂園」。

　
○  MUD 禮儀
　 　

　　隨著網路的普及化，MUD 再不是大學生的專利，一些高中/國中的學生、上班族都可有機會享受這項服務，但很多時他們卻忽略了應有的玩 MUD 禮節，玩 MUD 不是與生俱來的權力，雖說 MUD 是項免費的服務，但是設計者/管理者無條件的心力付出，免費提供消遣娛樂，是基於無為的服務之心，MUD 是需要玩家的協力維護。

　各 MUD 都有其獨有的限制及法律 (Rule)，玩家在進入之前必須詳細了解，如果覺得不適合自己，可以選擇別的 MUD 去玩，請不要惡意違反 MUD 的法律。在這個虛擬小社會中，人與人的接觸必然會產生磨擦，大家便要『忍一時風平浪靜，退一步海闊天空』，那些惡意搗亂的人，慢慢地會建立了不好的名聲，最後就會被歸類為『不受歡迎』的那群! 總之與人和諧相處及遵守遊戲規則，是玩 MUD 的最基本禮節。以下是一些玩家必須遵守的鐵規，一般的 MUD 站都適用。

　　◆ 不要在公共頻道重覆多次的同樣談話。
　　◆ 不要在公共頻道罵戰。
　　◆ 不要偷撿別人打死的怪物屍體內的裝備或金錢。
　　◆ 不要搶別人的怪物來打。
　　◆ 不要說髒話及淫穢話題。
